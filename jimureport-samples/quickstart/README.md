# 免安装启动版本

启动步骤：

- 1.本地安装jdk7+

- 2.下载项目 quickstart

```
    git clone https://gitee.com/jeecg/JimuReport.git
	cd JimuReport/jimureport-samples/quickstart
```

- 3.执行下面的脚本，初始化数据库

```
   quickstart/jimureport.mysql5.7.create.sql
```
   
- 4.双击 start.bat 启动

- 5.访问报表
    http://localhost:8085/jmreport/list
